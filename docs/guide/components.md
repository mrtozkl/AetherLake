# Components

The platform consists of several carefully curated open-source projects.

| Component | Role | Status |
|-----------|------|--------|
| **[MinIO](https://min.io/)** | S3-compatible object storage | ✅ Stable |
| **[Trino](https://trino.io/)** | Distributed SQL query engine | ✅ Stable |
| **[Apache Polaris](https://polaris.apache.org/)** | Iceberg REST Catalog | ✅ Stable |
| **[Apache Airflow](https://airflow.apache.org/)** | Workflow orchestration | ⚠️ Optional |
| **[Apache Spark](https://spark.apache.org/)** | Distributed data processing | ✅ Stable |
| **[Milvus](https://milvus.io/)** | Vector similarity search | ✅ Stable |
| **[Keycloak](https://www.keycloak.org/)** | Identity & SSO (OIDC) | ✅ Stable |
| **[dbt](https://www.getdbt.com/)** | SQL-based data transformation | ✅ Stable |
| **Control Panel** | Web UI for platform management | ✅ Stable |
